This Site

Despite what every member of the W3C would like you to believe, you (yes, you!) can just make your own website, and point a domain to someone that's hosting it (or host it yourself), and then anyone else can look at it. It's actually like completely amazing to me, even still after how much corporate interest has twisted it into something primarly used to make you (yes, you!) want to kill yourself, but that's neither here nor there!

This site is constructed solely of HTML and CSS written in either VSCodium or Neovim (depending on which computer I'm on when I feel like editing this site...)

I didn't have any experience with either of those languages outside of customizing my neopets pet-pages and myspace layout as a kid about 10,000 years ago, and learned it by just web searching whenever I wanted to do something. For a while, I was using eleventy as a static site generator, but managing the js dependencies across multiple machines was too freaking annoying despite how nice it was to just write each page in markdown, so umm ano eto bleh, I'm back just hand writing HTML again.

This site is hosted on NixiHost, which uses cPanel as its hosting platform. It was an absolutely massive learning curve compared to Neocities, which I was using before, but I am happy I went through the process of learning it :D. That said, if you're interested in making your own personal site, I would reccomend Neocities (as you've no doubt heard of), or maybe something like NekoWeb (which has a bit less restrictions on free accounts compared to Neocities). There's literally endless options and it's easy to learn and it's fun...
